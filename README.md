# WEBSITE THƯ VIỆN ĐIỆN TỬ TDMU

-   Xây dựng website quản lý thư viện cho Trường Đại học Thủ Dầu Một.

## Thành viên thực hiện

-   Trần Văn Hùng (Leader).
-   Trương Văn Toàn.

## Công nghệ sử dụng

-   ASP. NET MVC (.NET Framework), Bootstrap, JQuery(AJax).
